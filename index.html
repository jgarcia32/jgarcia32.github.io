Projects

 I along with a partner created an iOS game using apple's ARKit (augmented reality kit) and Swift. This was a semester long project for a course. We were given the freedom to create any application using any technologies and this is what we landed on. Neither of us worked with iOS, Swift, or ARKit before this project.
 The goal of the game is to stack blocks as high as possible while using your iPhones camera. The speed of the blocks increase with each placement. The parts of the blocks that hang off the tower fall off. Making the next foundation and moving block smaller. The game uses iOS CoreData to store highscores on each users personal device. I plan on putting this game on the appstore sometime soon.

The source code for this project can be found on my GitHub.

 As a personal project I decided to make a notes application that takes advantage of Google's cloud based service Firebase. This project is not complete as I am still working on it in my free time. The end goal was a project that is similar to Google Docs in that users can share a single Note and edit it at the same time.
 This is an iOS application that uses Swift and FirebaseAuth/Firestore. FirebaseAuth is a Google service that allows users to sign up and allows me to manage these users through the Firebase console online. Users can sign up through simple email and password or by using a google account/facebook/twitter/github etc. At the moment I only allow email and password but I plan to implement the rest.
 I use Firestore for the backend which is Google's RealTime Database. I can manage all the data online through the Firebase console. At the moment my application allows for user sign up/sign in and the ability to read from the database in realtime view. Whenever I add something to the database it loads in realtime to the application.

The source code for this project can be found on my GitHub.
